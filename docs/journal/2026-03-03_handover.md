## Key Decisions
- **z-proportional bandwidth over fixed**: `max(frac * z_spec, floor)` is physically motivated (photo-z errors scale with z) and configurable
- **Config placement**: Added to `PDFConfig` since these control PDF construction behavior
- **No tradeoff found**: Wider bandwidth improves all PDF metrics without degrading point estimates, confirming the baseline was severely under-dispersed
